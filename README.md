# Group Project – Weather App (Next.js + React)

## Project Overview
This is a simple weather application built using React through the Next.js framework.
The project follows the required structure: a Page component acting as the parent, and three child components: Header, Main, and Footer.
The app allows users to select a Canadian province and city, then displays weather information based on their selection.

## How to Run the Application Locally
1.	Install dependencies
2.	npm install
3.	Start the development server
4.	npm run dev
5.	Open the application
    http://localhost:3000
